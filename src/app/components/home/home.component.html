Navigation Erläuterung:

Lehrer/Fächer: Hier kann man den Lehrern weitere Fächer zuweisen, oder fehlende Lehrer ergänzen

Klassen/Fächer/Lehrer: Hier kann man die Anzahl der Stunden in einer Klasse für ein Fach festlegen und Lehrer diesem Fach der Klasse zuweisen

Gesamtplan: Hier kann man den Gesamt-Stundenplan aller Klassen ändern (Mo / Di / Mi / Do / Fr einzeln Navigation)

Epochenpläne: Hier kann man Termine ändern (Fahrten der Klassen, Projekte, Ferientermine etc.) und die Epoche / Schiene / Rhythmus bearbeiten (weiterhin Navigation Abschnittsweise bis zu den Ferien jeweils)

Vertretungsplan: Hier kann man die Vertretung planen und den Plan ausdrucken, es wird aber aktuell nichts gespeichert

Lehrerpläne: Hier kann von einzelnen Lehrern der Stundenplan angezeigt werden, oder per Button auch alle Lehrerpläne.
